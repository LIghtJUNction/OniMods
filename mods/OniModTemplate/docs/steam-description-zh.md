[h1]OniModTemplate[/h1]

[b]Oxygen Not Included Mod 模板项目[/b]

[h2]用途[/h2]
[list]
[*] 为快速创建新 Mod 提供最小可运行模板。
[*] 适用于 `onim init` 后的二次开发、补丁编写与发布流程。
[*] 已预置基础目录结构与构建链路。
[/list]

[h2]特性[/h2]
[list]
[*] 提供可直接运行的模组工程骨架。
[*] 包含 `onim` 构建系统常用布局与基础 `ModInfo` 入口。
[*] 适合学习、验证、快速脚手架。
[/list]

[h2]适用对象[/h2]
[list]
[*] 需要快速搭建新 Mod 的开发者。
[*] 想要复用已有工程结构和发布节奏的项目。
[*] ONI 插件实验和教学示例场景。
[/list]
