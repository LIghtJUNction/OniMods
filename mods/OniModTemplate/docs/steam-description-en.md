[h1]OniModTemplate[/h1]

[b]Oxygen Not Included mod template project[/b]

[h2]Purpose[/h2]
[list]
[*] A minimal starter template for creating ONI mods quickly.
[*] Intended for teams who need a clean scaffold before writing gameplay patches.
[*] Ready for `onim`-based build and publish workflows.
[/list]

[h2]Features[/h2]
[list]
[*] Predefined project layout suitable for quick iteration.
[*] Basic `ModInfo` entry point structure included.
[*] Useful for learning, testing, and bootstrap development.
[/list]

[h2]Who this is for[/h2]
[list]
[*] Mod developers building a new ONI project from scratch.
[*] Users needing a consistent scaffold for patches and experiments.
[*] Teams that want a predictable starting layout.
[/list]
